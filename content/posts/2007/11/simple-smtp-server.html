--- url: /2007/11/simple-smtp-server.html title: "Simple SMTP server" date: "2007-11-17T07:28:00.002+01:00" tags: [ EEM ] --- When I was testing the SMTP client on the routers (configured with action â€¦ mail command or as part of EEM SMTP library), I didn't want the messages with weird addresses and content to circulate through our e-mail system (plus I wanted to see the results immediately), so I wrote a simple SMTP server in Perl that prints the messages it receives. It accepts a single parameter: the IP address to listen on (only needed on workstations with multiple IP addresses).

You need the Net::SMTP::Server module on top of the standard Perl distribution to make it work.

You can download the PERL source code or compiled EXE file (for Windows users) from CT3 wiki.
