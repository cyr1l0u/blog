--- title: "Introduction to Embedded Menu Manager" date: "2008-08-19T06:48:00.000+02:00" tags: [ network management,command line interface,EMM ] --- One of the great new features introduced in IOS release 12.4(20)T is the Embedded Menu Manager (EMM), which allows you to build dynamic hierarchical menus with a combination of static definitions in XML and Tcl scripts. The Cisco documentation on this feature is quite cryptic, so I wrote an introduction to EMM, describing the major XML elements and the ways to use it.

Read more in CT3 wiki.
