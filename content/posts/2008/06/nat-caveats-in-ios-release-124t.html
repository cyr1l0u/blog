--- url: /2008/06/nat-caveats-in-ios-release-124t.html title: "NAT caveats in IOS release 12.4T" date: "2008-06-12T06:50:00.001+02:00" tags: [ NAT ] --- If you have upgraded your router from any other IOS release to release 12.4T without changing the NAT configuration (or used NAT configuration known to work into a router running IOS release 12.4T), you might have encountered weird behavior due to the changes in NAT implementation. The unexpected behavior and configuration fixes needed to avoid the NAT-related problems are described in my new NAT Caveats in IOS release 12.4T article.
