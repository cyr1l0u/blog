--- title: "Conditional BGP route origination" date: "2008-05-28T07:09:00.002+02:00" tags: [ BGP ] --- Sebastian Majewski has found an interesting feature: if you use the network route-map BGP configuration command to originate BGP prefixes and use the match conditions within the route-map, BGP inserts the IP prefix in the BGP table only if the source route in the IP routing table satisfies the route-map conditions. More details and a sample router configuration are available in CT3 wiki.
