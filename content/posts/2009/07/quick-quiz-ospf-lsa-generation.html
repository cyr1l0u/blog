Given the following OSPF configuration …

interface Loopback 0
 ip ospf 1 area 0
!
interface Ethernet 0
 ip ospf 1 area 0
!
interface Serial 1
 ip ospf 1 area 0
!
router ospf 1
 passive-interface Ethernet 0

… how many LSAs will the router originate? Leave your opinions in the comments.
